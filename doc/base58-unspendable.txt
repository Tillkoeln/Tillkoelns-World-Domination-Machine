This is a list of unspendable addresses with which to burn coins to. 
When you send a coin to any of these addresses, they are permanently gone and can never be redeemed.
You can prove this by sending a small test amount to the address
 and then checking a block explorer or using getrawtransaction txid 1.
 You should see the coins output to somehting like 
 "raw_scriptPubKey": "76a914000000000000000000000000000000000000000088ac" 
 Some block explorers are also smart enough to indicate in their user interface that the coins were destroyed.
 
 You can find the version number matching to the desired altcoin by checking two places in the code:

chainparams.cpp for bitcoin 0.9 and later based coins:

base58Prefixes[PUBKEY_ADDRESS] = list_of(0); //version number is 0

or in base58.h

PUBKEY_ADDRESS = 0, //version number is 0
